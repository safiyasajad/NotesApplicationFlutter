## About the Project

This project is a simple notes application built with Flutter.

Users can:

- Create notes
- Edit notes
- Update notes
- Delete notes

The app uses **SQLite** as an offline local database.

## Tools & Technologies

- Flutter
- SQLite
- Android Studio
- Android Emulator

✅ Offline storage
