# notesapp

This is my first mobile application project
This is a demo project that I learnt from Flutter Agentic's youtube channel "https://www.youtube.com/watch?v=vkbHytGU_oU"
This project is a notes app where data can be inserted into, edited, updated and deleted.
It used the offline database sqlite
It requires the use of an emulator. I used android studio to set up the enviornment for the emulator.
